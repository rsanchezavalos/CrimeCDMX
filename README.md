# CrimenCDMX

An interactive visualization of Mexico City's crime data using Shiny applications and R

### Prerequisites


```
- rgdal
- ggplot2
- ggmap
- downloader
- knitr
- tidyverse
- leaflet
- RColorBrewer
- scales
- lattice
```

## Deployment

```
shiny::runApp('workspace/CrimeCDMX')
```
